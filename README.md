# Banking Data Engineering Portfolio (SQL / PL/SQL / Python)

This repository contains 5 small projects designed to demonstrate practical banking data engineering and production support skills.

## Projects
1. **SQL Lineage Extraction** (`01-sql-lineage-extraction`)
2. **ETL Part 1: Extract + Validate** (`02-etl-extract-validate`)
3. **ETL Part 2: Transform + Performance + Validate** (`03-etl-transform-performance`)
4. **Banking Data Reconciliation** (`04-banking-reconciliation`)
5. **Production Monitoring + Alerting** (`05-prod-monitoring-alerting`)

## How to review quickly
Each project folder contains:
- `README.md` with requirements and expected output
- `src/` source code
- `data/` sample data (synthetic)
- `output/` example output
